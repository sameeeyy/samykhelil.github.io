PointCloudFR - Plugin QGIS pour données LiDAR

Janvier 2025

Développement de PointCloudFR, un plugin open-source QGIS qui simplifie l'accès aux données LiDAR nationales françaises. Cet outil permet aux professionnels SIG et chercheurs de télécharger et traiter efficacement des données de nuages de points depuis l'IGN (Institut National Géographique).

Points clés :

  • Conception d'algorithmes intelligents de sélection de tuiles avec trois stratégies de traitement
  • Intégration complète avec QGIS : visualisation automatique des nuages de points
  • Implémentation d'une architecture de téléchargements simultanés des tuiles LiDAR
  • Développement d'un système de gestion des erreurs et de dépendances automatiques pour QGIS
  • Déploiement dans le répertoire officiel des plugins QGIS (+ 2000 téléchargements)

Technologies utilisées :

Python, API QGIS, Framework Qt, Github

Analyse de l'imperméabilisation des sols par IA

2024

Développement d'un modèle d'intelligence artificielle pour analyser l'imperméabilisation des sols et ses impacts sur la surchauffe urbaine à partir d'images satellitaires.

Points clés :

  • Développement d'un modèle CNN (U-Net) pour la segmentation d'images satellitaires
  • Classification automatique des surfaces perméables et imperméables
  • Analyse de l'évolution des températures de surface par imagerie satellitaire
  • Démonstration d'une augmentation de 6°C liée à l'imperméabilisation

Technologies utilisées :

Python (TensorFlow, Numpy, Rasterio), QGIS
